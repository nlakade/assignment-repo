Setup Instructions
For LinkedIn Scraper:
Navigate to linkedin_scraper folder

Run bundle install

Install ChromeDriver

Update the script with test LinkedIn credentials

Run ruby scraper.rb

For Autodialer App:
Navigate to autodialer_app folder

Run bundle install

Run rails db:create db:migrate

Run rails server

Visit http://localhost:3000

For Blog Generator:
Navigate to blog_generator folder

Run bundle install

Run rails db:create db:migrate

Run rails server

Visit http://localhost:3000
